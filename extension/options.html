Biligumi Connector 插件版

页面内设置

打开 Bilibili 视频页或番剧播放页后，Biligumi 面板会提供 Token、白名单、绑定、同步、角色横栏、条目信息栏、自动标记已看和 OP/ED 跳过设置。

浏览器快捷键

默认快捷键：Alt+Shift+Right。当前浏览器分配：读取中...。

Chrome 打开 chrome://extensions/shortcuts 修改；Edge 打开 edge://extensions/shortcuts 修改。

安装后检查

  • 扩展需要允许访问 Bilibili、Bangumi API 和 bgm.tv。
  • 油猴版和插件版建议二选一启用，避免重复面板。
  • 插件版数据保存在 chrome.storage.local，不会自动读取油猴管理器里的旧数据。
